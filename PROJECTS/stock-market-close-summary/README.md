# A股收盘总结任务

## 任务概述

执行时间：每天 15:10（A股下午收盘后）

任务内容：
1. 使用 akshare 获取A股全天行情数据（带重试机制）
2. 分析大盘走势和涨跌家数
3. 全天热点板块分析
4. 北向资金流向
5. 使用 akshare 获取（多数据源备份）：
   - 涨幅超前股票（前5只）- 失败重试3次
   - 跌幅较大股票（前5只）- 失败重试3次
   - 活跃股票（成交量前5）- 失败重试3次
6. 明日展望
7. 发送到钉钉群

## 数据获取优化

- 重试机制：失败自动重试3次
- 多数据源：东方财富+同花顺备份
- 延迟获取：15:10执行（收盘后5分钟）
- 容错处理：部分失败也发送报告
