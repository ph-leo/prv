### 2.1 Prompt优化（核心方案）

#### 2.1.1 骨折专用Prompt设计

**问题诊断**：
- 当前Prompt过于泛化，缺乏骨折识别指导
- 没有针对不同部位骨骼的检查清单
- 缺少高风险漏诊骨折的提醒

**优化方案**：

```python
# 骨折专用Prompt模板
FRACTURE_PROMPT_V2 = """
# 骨折识别专用Prompt (SKELETAL/X-RAY)

## 🔍 骨折识别步骤

### Step 1: 骨皮质检查（关键）
1. 仔细检查每块骨骼的骨皮质是否连续
2. 寻找骨皮质断裂、不连续、移位
3. 特别关注小骨骼（指骨、趾骨、小腕骨）的细微骨折线

### Step 2: 骨密度检查
1. 寻找骨密度异常区域
2. 检查是否有骨压缩、塌陷
3. 注意骨小梁排列是否紊乱

### Step 3: 骨轮廓检查
1. 骨骼轮廓是否规整
2. 是否有成角、移位、旋转
3. 关节间隙是否正常

## 📋 骨折检查清单（必须逐项检查）

### 上肢骨骼
- [ ] 锁骨：内外侧端、中段
- [ ] 肩胛骨：肩峰、喙突、肩胛颈
- [ ] 肱骨：外科颈、骨干、内上髁、外上髁
- [ ] 桡骨：远端（腕关节处）、骨干、尺骨茎突
- [ ] 尺骨：鹰嘴、骨干、桡骨茎突
- [ ] 掌骨：各掌骨骨干、基底部、头
- [ ] 指骨：各指骨骨干、基底部、末节

### 下肢骨骼
- [ ] 髋骨：髂骨、坐骨、耻骨
- [ ] 股骨：股骨颈、骨干、大转子、小转子、内外髁
- [ ] 胫骨：骨干、内外踝、髁间隆起
- [ ] 腓骨：骨干、内外踝
- [ ] 距骨、跟骨、舟骨、楔骨、骰骨
- [ ] 跖骨：各跖骨骨干、基底部、头
- [ ] 趾骨：各趾骨骨干、基底部、末节

### 脊柱
- [ ] 椎体：各椎体前缘、后缘、上终板、下终板
- [ ] 椎弓：椎板、棘突、横突
- [ ] 关节突关节
- [ ] 椎间隙高度

## 🎯 骨折典型特征描述

| 骨折类型 | 影像特征 | 常见部位 | 识别要点 |
|---------|---------|---------|---------|
| 线性骨折 | 单一透亮线，边缘清晰 | 长骨骨干 | 关注透亮线，对比两侧 |
| 粉碎性骨折 | 多个骨碎片，碎裂影 | 股骨、胫骨、跟骨 | 寻找多个骨折线 |
| 嵌插性骨折 | 骨密度增高，骨皮质皱褶 | 股骨颈、肱骨外科颈 | 关注密度增高区域 |
| 压缩性骨折 | 椎体高度降低，前缘变平 | 胸腰椎 | 测量椎体高度 |
| 骨骺分离 | 骨骺与干骺端移位 | 儿童长骨 | 对比双侧骨骼 |

## ❗ 高风险漏诊骨折提醒（重点）

### 1. 隐匿性骨折（最容易漏诊）
**特征**：骨皮质断裂但无明显移位，密度差异小

**重点部位**：
- 耻骨支、坐骨支（骨盆）
- 小腕骨（舟骨、月骨）
- 距骨、跟骨（足部）
- 指骨、趾骨（手足小骨）

**扫描策略**：
- 仔细检查骨皮质连续性
- 对比对侧相同部位
- 关注局部骨密度变化

### 2. 应力性骨折
**特征**：早期无移位，仅有骨皮质不规则或斑片状密度增高

**重点部位**：
- 第二、三跖骨（足部）
- 胫骨（下段）
- 腓骨（下段）

**扫描策略**：
- 对比双侧骨骼形态
- 关注骨皮质是否规则
- 注意局部骨膜反应

### 3. 嵌插性骨折
**特征**：骨皮质不完整但无明显移位，局部密度增高

**重点部位**：
- 股骨颈（老年人常见）
- 肱骨外科颈
- 椎体（压缩性骨折）

**扫描策略**：
- 测量椎体高度（对比正常值）
- 关注局部骨密度是否增高
- 注意骨皮质是否皱褶

## 📝 报告输出格式（强制要求）

### 必须包含的字段：

```
FINDINGS:

1. **骨折部位**：具体骨骼名称 + 骨折部位（近端/中段/远端）
   - 例如：右锁骨远端1/3

2. **骨折类型**：根据上述分类明确骨折类型
   - 例如：粉碎性骨折、线性骨折、嵌插性骨折

3. **骨折程度**：轻微/中度/重度（基于移位程度）
   - 轻微：无移位或轻度移位
   - 中度：中度移位，成角<30°
   - 重度：严重移位，成角≥30°或旋转

4. **是否移位**：是/否（如有移位，描述移位方向）
   - 例如：骨折端分离约3mm，向上成角15°

5. **并发症**：是否伴有关节脱位、软组织损伤
   - 关节脱位：是/否
   - 软组织损伤：是/否（描述损伤程度）

6. **置信度**：高/中/低
   - 高：骨折特征明显
   - 中：可疑骨折，需要进一步评估
   - 低：骨折特征不典型，可能性小
```

### 示例输出（骨折病例）：

```
FINDINGS:

1. **右侧第5-8肋骨骨折**（线性骨折，伴移位）
   - 位置：右侧第5-8肋骨（前外侧）
   - 类型：线性骨折，部分骨皮质连续性中断
   - 程度：中度（骨折端分离约2-3mm）
   - 移位：是，骨折端轻度分离
   - 并发症：无明显气胸或血胸
   - 置信度：高
```

## ⚠️ 注意事项

1. **不要漏检小骨**：指骨、趾骨、小腕骨、小跗骨
2. **注意隐匿性骨折**：无移位的骨折容易被忽略
3. **对比双侧**：对于对称部位，对比双侧形态
4. **关注骨密度**：嵌插性骨折表现为密度增高
5. **量化移位**：尽可能描述移位程度（mm或角度）

判断该X光片是否显示骨折。如果有骨折，请详细描述骨折的位置、类型、程度和移位情况。
"""
```

#### 2.1.3 Prompt版本控制

```python
# 代码中需要维护多个Prompt版本
PROMPT_VERSIONS = {
    "v1.0": "Compare this XXX X-ray to normal anatomy: ...",  # 原始Prompt
    "v2.0-fracture": FRACTURE_PROMPT_V2,  # 骨折专用Prompt
    "v2.0-pulmonary": PULMONARY_PROMPT_V2,  # 肺部专用Prompt
    "v2.0-general": GENERAL_PROMPT_V2,  # 综合Prompt
}

def get_prompt(model_type, image_type):
    """根据模型类型和影像类型选择合适的Prompt"""
    if model_type == "fracture":
        return PROMPT_VERSIONS["v2.0-fracture"].replace("XXX", image_type)
    elif model_type == "pulmonary":
        return PROMPT_VERSIONS["v2.0-pulmonary"].replace("XXX", image_type)
    else:
        return PROMPT_VERSIONS["v2.0-general"].replace("XXX", image_type)
```

PROMPT_VERSIONS = {
    "v1.0": "...",  # 原始Prompt
    "v2.0-fracture": "...",  # 骨折专用Prompt
    "v2.0-pulmonary": "...",  # 肺部专用Prompt
}

def get_prompt_v2(image_type, lesion_type):
    """根据影像类型和病种类型选择Prompt"""
    if lesion_type == "fracture":
        return PROMPT_VERSIONS["v2.0-fracture"].replace("XXX", image_type)
    elif lesion_type in ["pneumonia", "atelectasis", "effusion"]:
        return PROMPT_VERSIONS["v2.0-pulmonary"].replace("XXX", image_type)
    else:
        return PROMPT_VERSIONS["v2.0-general"].replace("XXX", image_type)

def analyze_image_v2(image_path, patient_info):
    """V2.0分析流程"""
    # 1. 读取影像
    dicom = pydicom.dcmread(image_path)
    
    # 2. 预处理
    processed = preprocess_clahe(dicom)
    
    # 3. 病种识别（自动判断）
    lesion_type = detect_lesion_type(dicom, patient_info)
    
    # 4. 选择Prompt
    prompt = get_prompt_v2(dicom.Modality, lesion_type)
    
    # 5. 调用AI API
    result = call_ai_api(prompt, processed)
    
    # 6. 解析结果
    parsed_result = parse_result_v2(result)
    
    return parsed_result
