## 📋 工作流程

### 1️⃣ 测试启动阶段
```
测试工程师 → 查看 职责清单.md
测试工程师 → 查看 README.md
测试工程师 → 运行 quick_test.sh
测试工程师 → 开始测试
```

### 2️⃣ 测试执行阶段
```
测试工程师 → 每小时更新 进度报告模板.md
测试工程师 → 遇到问题 → 填写 问题汇报模板.md
极光 → 每小时检查 进度报告
极光 → 提供协助（如果需要）
```

### 3️⃣ 问题处理阶段
```
测试工程师 → 填写 问题汇报模板.md
测试工程师 → 发送给 极光
极光 → 分析问题 → 提供解决方案
极光 → 如果无法解决 → 报告 负责人
```

### 4️⃣ 验收阶段
```
测试工程师 → 完成测试
极光 → 生成验收报告
极光 → 提交给 负责人
负责人 → 审核确认
```
