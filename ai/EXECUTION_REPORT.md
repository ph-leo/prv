## 一、优化方案实施详情

### 1. 骨折检测 Prompt 增强 (`fracture_prompt_v2_opt.py`)

#### 优化点：

1. **强化骨皮质连续性检查**
   - 明确要求逐段检查骨皮质是否连续、平滑
   - 寻找骨皮质中断、扭曲、不规则等异常征象
   - 注意骨皮质边缘是否清晰、整齐

2. **添加高风险漏诊骨折提醒**
   - 隐匿性骨折（Occult Fracture）：X光早期可能阴性，需结合CT或MRI
   - 应力性骨折（Stress Fracture）：常见于运动员，早期表现为骨膜反应
   - 嵌插性骨折（Impaction Fracture）：骨折端嵌入，骨皮质可能保持完整
   - 青枝骨折（Greenstick Fracture）：儿童常见，骨皮质部分断裂
   - 骨骺分离（Epiphyseal Separation）：儿童生长板损伤

3. **添加详细的上肢/下肢/脊柱检查清单**
   - **上肢检查**：肩关节、上臂、前臂、手部（特别强调舟骨）
   - **下肢检查**：骨盆、大腿、膝关节、小腿、足部
   - **脊柱检查**：颈椎、胸椎、腰椎、骶尾骨

4. **强制报告格式**
   - 【骨折诊断报告】标题
   - 部位定位
   - 影像学表现（骨皮质、骨折线、移位、成角、旋转）
   - 骨折类型
   - 鉴别诊断（3-5个）
   - 诊断建议（CT/MRI检查）
   - 风险提示（高风险漏诊提醒）

### 2. 肺部病变检测 Prompt 增强 (`pulmonary_prompt_v2_opt.py`)

#### 优化点：

1. **细化肺部病变分类**
   - **胸膜病变**：气胸、胸腔积液
   - **肺实质病变**：肺炎、肺不张、肺气肿
   - **其他重要病变**：肺结核、肺癌、肺栓塞

2. **添加密度、边缘、分布特征描述**
   - **密度特征**：较低/正常/较高密度，均匀/不均匀
   - **边缘特征**：清晰/模糊，规则/不规则，分叶/毛刺/脊状
   - **分布特征**：单侧/双侧，局灶/多发/弥漫，肺野外带/肺门周围/纵隔旁

3. **添加高风险漏诊病变提醒**
   - 气胸：张力性气胸（紧急情况）、闭合性、医源性
   - 积液：少量积液、包裹性、血胸
   - 肺不张：代偿性肺气肿、小量肺不张
   - 肺癌：周围型、小结节（≤1cm）、隐匿性
   - 肺栓塞：周围型肺梗死、肺动脉高压

4. **强制报告格式**
   - 【胸部影像诊断报告】标题
   - 影像学表现（部位、性质、密度、边缘、分布、伴随）
   - 诊断意见（主诊断+3个鉴别诊断）
   - 建议（进一步检查、随访、临床_correlation）

### 3. 通用 Prompt 增强 (`generic_prompt_v2_opt.py`)

#### 优化点：

1. **添加低识别率病灶的专门提示**
   - 微小结节（≤5mm）：建议短期随访（3个月）
   - 磨玻璃影（GGO）：建议CT随访（6-12个月）
   - 纵隔淋巴结（≤10mm）：结合临床、PET-CT
   - 胸膜下病变：薄层CT、增强扫描
   - 心后区病变：侧位片、CT扫描
   - 膈顶病变：呼气相、CT扫描
   - 骨皮质内微小病变：骨窗、MRI
   - 血管旁微小结节：增强扫描
   - 脊柱旁病变：冠状/矢状重建
   - 肺尖病变：侧位片、CT薄层扫描

2. **强化影像征象描述要求**
   - 密度/信号特征（CT/MRI/X光差异）
   - 边缘特征（清晰/模糊/规则/不规则/分叶/毛刺）
   - 分布特征（位置、数量、形态）
   - 内部结构（实性/囊性/混合/钙化）
   - 增强特征（强化方式/程度/时间）

3. **添加多视角检查提醒**
   - **胸部影像**：正位、侧位、侧斜位、高千伏、呼气/吸气相
   - **CT影像**：平扫、增强、薄层重建（≤1mm）、MPR重建、VR重建
   - **MRI影像**：T1WI、T2WI、脂肪抑制、增强扫描

4. **强制报告格式**
   - 【影像诊断报告】标题
   - 影像学表现（10项详细描述）
   - 诊断意见（4项：主诊断+3个鉴别诊断）
   - 建议（4项：进一步检查、随访、临床_correlation、多学科会诊）
