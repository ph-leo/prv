# A股模拟操盘系统

这是一个专业的A股模拟交易系统，提供开盘信号扫描、智能强度评级和10天模拟交易周期。

## 核心功能

### 1. 模拟账户管理
- 初始化5万元模拟资金
- 跟踪持仓、现金、总市值
- 实时计算盈亏情况

### 2. 开盘数据获取
- 使用 akshare 获取A股实时数据
- 支持沪深京A股全市场扫描
- 实时行情、涨跌幅、成交量等指标

### 3. S1级别信号识别
- **S1强度评级 (≥95%)**: 信号强度极高，推荐交易
- **S2强度评级 (≥85%)**: 信号强度高，考虑交易
- **S3强度评级 (≥70%)**: 信号强度中等，观察即可

### 4. 交易执行流程
1. 每日09:30扫描开盘信号
2. 发现S1级别信号后发送钉钉群
3. 等待用户确认（执行/跳过）
4. 根据确认执行模拟交易
5. 10天后生成盈亏报告

## 技术栈

- **数据源**: akshare (Python股票数据库)
- **交易时间**: A股 09:30-11:30, 13:00-15:00
- **信号频率**: 每日开盘扫描
- **模拟周期**: 10个交易日
- **_signal strength algorithm_: Multivariate approach (4 dimensions)
  - 涨跌幅权重: 40%
  - 成交量权重: 30%
  - 价格位置权重: 20%
  - 稳定性权重: 10%

## 使用场景

- 🔍 定时任务集成：每天09:30自动扫描

## 信号强度算法

系统使用多变量算法计算股票信号强度 (0-100%)：

1. **涨跌幅** (40%): 收益越高，强度越大
2. **成交量** (30%): 成交额越大，热度越高
3. **价格位置** (20%): 高价股往往更强
4. **稳定性** (10%): 价格越稳定，信号越可靠

强度分级:
- S1 (最强): ≥95% - 信号强度极高，推荐交易
- S2 (强): ≥85% - 信号强度高，可以考虑
- S3 (中): ≥70% - 信号强度中等，建议观察
- D (弱): <70% - 信号较弱，建议观望

## 安全提示

⚠️ **重要提醒**:
1. 本系统为模拟交易，不涉及真实资金
2. 信号基于历史数据和技术指标，不构成投资建议
3. 模拟交易不考虑印花税、手续费等交易成本

## 配置

在 Gateway 中添加定时任务：
```yaml
cron:
  - name: trading-simulation-50k
    schedule:
      kind: every
      everyMs: 86400000  # 每天一次
    payload:
      kind: systemEvent
      text: "【A股模拟操盘 - 5万元】"
    sessionTarget: main
```
