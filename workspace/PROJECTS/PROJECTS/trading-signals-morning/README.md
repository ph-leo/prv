# A股盘前信号扫描机器人 - 使用说明

## 项目信息

**任务ID**: `a1b2c3d4-e5f6-7890-abcd-ef1234567890`  
**任务名称**: trading-signals-morning  
**执行时间**: 每天 08:30（A股开盘前）

## 功能说明

1. **获取隔夜美股数据** - 使用 akshare 获取美股指数数据
2. **分析对A股开盘的影响** - 分析美股涨跌幅，判断对A股开盘的影响
3. **预判热点板块** - 基于资金流数据预判今日热门板块
4. **推荐股票** - 筛选5只今日推荐股票（资金流入、技术形态良好）
5. **发送到钉钉群** - 将报告发送到指定钉钉群

## 使用方法

### 环境依赖

```bash
pip install akshare pandas numpy requests
```

### 运行方式

```bash
cd /root/.openclaw/workspace/PROJECTS/trading-signals-morning
source venv/bin/activate
python main.py
```

### 配置钉钉推送

设置环境变量 `DINGTALK_WEBHOOK` 为钉钉机器人Webhook地址：

```bash
export DINGTALK_WEBHOOK="https://oapi.dingtalk.com/robot/send?access_token=xxx"
python main.py
```

## 实现说明

### 数据源

- **美股数据**: akshare 的 `index_us_stock_sina` 接口
- **板块数据**: akshare 的 `stock_fund_flow_concept` 接口
- **个股数据**: akshare 的 `stock_individual_fund_flow_rank` 接口

### 网络错误处理

- 网络请求失败时使用重试机制（最多3次）
- 如果所有接口均失败，使用默认数据

## 定时任务配置

如需配置每天08:30自动执行，可添加 crontab：

```bash
crontab -e
```

添加：

```
30 8 * * * /root/.openclaw/workspace/PROJECTS/trading-signals-morning/run.sh
```

## 注意事项

1. 网络连接问题可能导致部分数据接口无法获取，程序会自动切换到备用数据
2. 钉钉推送需要配置 `DINGTALK_WEBHOOK` 环境变量
3. 如需修改发送目标钉钉群，请修改 `DINGTALK_WEBHOOK` 环境变量
