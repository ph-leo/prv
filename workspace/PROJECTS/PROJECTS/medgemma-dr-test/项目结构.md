## 📁 详细目录说明

### 📁 /e/ai_test_MedGemma/ai/other/
**用途**: 项目根目录  
**说明**: 所有测试相关的文件都在此目录下

### 📁 /e/ai_test_MedGemma/ai/other/scripts/
**用途**: 测试脚本目录  
**包含**: 
- `full_test_v7.4_300cases.py` - 主测试脚本

### 📁 /e/ai_test_MedGemma/ai/other/testfile300-20260312/
**用途**: 测试数据目录  
**包含**:
- `测试记录20260312-300.xlsx` - Excel 输入文件
- `test_study300/` - DICOM 原始数据目录

### 📁 /e/ai_test_MedGemma/ai/other/testfile300-20260312/test_study300/
**用途**: DICOM 原始数据  
**说明**: 300 个病例的 DICOM 文件，**禁止复制、上传、外传**  
**命名**: 每个病例一个文件夹，以病例 ID 命名

### 📁 /e/ai_test_MedGemma/ai/other/converted_images_v74/
**用途**: 图像转换结果  
**说明**: DICOM 转换为 PNG 并应用 CLAHE 增强  
**结构**: 与 `test_study300` 对应的文件夹结构

### 📁 /e/ai_test_MedGemma/ai/other/test_results/
**用途**: 测试结果目录  
**说明**: 测试完成后生成的报告和结果

### 📁 /e/ai_test_MedGemma/ai/other/test_results/v7.4_300cases/
**用途**: V7.4 测试结果  
**包含**:
- `summary_report_YYYYMMDD_HHMMSS.md` - Markdown 总报告
- `280469649_result.json` - 单个病例结果 JSON
- ... (300 个病例结果)
