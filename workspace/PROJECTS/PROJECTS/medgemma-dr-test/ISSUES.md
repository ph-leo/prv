# MedGemma V7.4 问题记录

## 当前问题

### #1 SSH 隧道自动重连受限

**状态**: 🔴 待解决
**发现时间**: 2026-04-03 21:21 CST

#### 问题描述

SSH 隧道已断开，无法自动重连，因为目标服务器启用了双重认证：
1. 密码认证（可自动）
2. 动态密钥/2FA（需要人工输入）

#### 影响

- 无法自动运行 300 例测试
- 需要人工介入重新建立隧道

#### 可能的解决方案

1. **短期方案**: 人工重新建立隧道，然后运行测试
2. **中期方案**: 配置 SSH 密钥认证，跳过密码输入
3. **长期方案**: 在开发电脑上部署自动化代理，减少隧道依赖

#### 行动项

- [ ] 人工重新建立 SSH 隧道
- [ ] 评估配置 SSH 密钥认证的可行性
