# MedGemma V7.4 SSH 配置

## 连接信息

| 参数 | 值 |
|------|-----|
| 跳板机 | fang@221.7.221.10:51622 |
| 目标电脑 | 10.10.30.39:22 (内网) |
| 本地端口 | 127.0.0.1:12222 |

## 连接命令

```bash
# 建立隧道
ssh -N -L 12222:10.10.30.39:22 -p 51622 fang@221.7.221.10

# 连接开发电脑
ssh -p 12222 D@127.0.0.1
```

## 认证方式

- **第一重**: 密码认证 (固定密码)
- **第二重**: 动态密钥 (2FA/OTP) ⚠️ 需要人工输入

## 重要规则

⚠️ **保持长期运行，不要关闭**

## 断开处理

如隧道断开，需要人工介入重新建立连接（需要动态密钥）。

## 脚本位置

- `/root/ssh-proxy/dev_pc` - 隧道启动脚本
- `/root/ssh-proxy/check_and_start_tunnel.sh` - 自动检查启动脚本
