### 优化要点

#### 1. 骨折专用Prompt (fracture_prompt_v2.py)

**核心改进**：
- 强化骨皮质连续性检查（必须逐段检查）
- 添加高风险漏诊骨折类型提醒（隐匿性、应力性、嵌插性）
- 详细的检查清单（上肢/下肢/脊柱）
- 强制报告格式（部位、类型、程度、移位、并发症）

**期望提升**：骨折识别率从 62.5% 提升至 75-80%

#### 2. 肺部专用Prompt (pulmonary_prompt_v2.py)

**核心改进**：
- 细化肺部病变分类（肺炎、肺不张、肺气肿等）
- 添加密度、边缘、分布特征描述
- 高风险漏诊病变提醒（隐匿性肺炎、小量积液等）
- 强制报告格式（位置、性质、大小、密度、边界、伴随征象）

**期望提升**：积液识别率从 55.1% 提升至 65-70%

#### 3. 通用增强Prompt (generic_prompt_v2.py)

**核心改进**：
- 多视角检查提醒（X光/CT/MRI）
- 低识别率病灶专项提示（微小结节、磨玻璃影等）
- 影像征象描述标准（密度、边缘、分布）
- 强制报告格式（部位、性质、大小、数量、形态、内部结构）

## ⚙️ 测试执行步骤

### 第一步：启动 SSH 隧道（用户操作）

用户需要在自己的开发电脑上执行以下命令：

```bash
# 在开发电脑上启动 SSH 隧道
cd /d/ssh-proxy  # 或相应路径
nohup ./dev_pc > /tmp/dev_pc.log 2>&1 &

# 等待隧道启动
sleep 3

# 验证隧道是否工作
ssh -p 12222 D@127.0.0.1 "hostname && whoami"
```

**注意**: SSH 连接需要动态密钥认证，请准备好输入动态密钥。
