## 二、识别率瓶颈定位

### 2.1 当前Prompt分析

当前Prompt存在以下问题：

#### 问题1: Prompt过于泛化
```
当前Prompt: "Compare this XXX X-ray to normal anatomy"
```
- 缺乏具体的病种识别指导
- 没有提供常见病变的特征描述
- 缺少分类学指导

#### 问题2: 缺少多阶段分析
- 只有单一Prompt，没有分步分析
- 缺少从宏观到微观的层层深入
- 无法处理复杂混合病变

#### 问题3: 缺少正向提示
- Prompt主要描述"NORMAL vs ABNORMAL"
- 缺少具体病变的识别关键词
- 缺少解剖结构的定位指导

## 三、优化方案

### 3.1 方案一：Prompt优化（推荐优先级：高）

#### 3.1.1 骨折专用Prompt（预计提升：15-20%识别率）

**核心思路**：
- 强化骨折识别关键词
- 添加骨折类型分类
- 添加骨折定位指导

**优化Prompt示例**：
```markdown
# 骨折识别专用Prompt (SKELETAL/X-RAY)

## 🔍 骨折识别步骤

### Step 1: 骨皮质检查
1. 检查每块骨骼的骨皮质是否连续
2. 寻找骨皮质断裂、不连续、移位
3. 注意小骨骼（指骨、趾骨、小腕骨）的细微骨折

### Step 2: 骨密度检查
1. 寻找骨密度异常区域
2. 检查是否有骨压缩、塌陷
3. 注意骨小梁排列是否紊乱

### Step 3: 骨轮廓检查
1. 骨骼轮廓是否规整
2. 是否有成角、移位、旋转
3. 关节间隙是否正常

## 📋 骨折识别清单（检查项）

- [ ] 锁骨：内外侧端、中段
- [ ] 肩胛骨：肩峰、喙突、肩胛颈
- [ ] 肱骨：外科颈、骨干、内上髁
- [ ] 桡骨：远端、骨干、尺骨茎突
- [ ] 尺骨：鹰嘴、骨干、桡骨茎突
- [ ] 掌骨：各掌骨骨干、基底部
- [ ] 指骨：各指骨骨干、基底部、末节
- [ ] 髋骨：髂骨、坐骨、耻骨
- [ ] 股骨：股骨颈、骨干、内外髁
- [ ] 胫腓骨：骨干、内外踝
- [ ] 足骨：跗骨、跖骨、趾骨

## 🎯 骨折典型特征描述

| 骨折类型 | 影像特征 | 常见部位 |
|---------|---------|---------|
| 线性骨折 | 单一透亮线，边缘清晰 | 长骨骨干 |
| 粉碎性骨折 | 多个骨碎片，碎裂影 | 股骨、胫骨 |
| 嵌插性骨折 | 骨密度增高，骨皮质皱褶 | 股骨颈 |
| 压缩性骨折 | 椎体高度降低，前缘变平 | 胸腰椎 |
| 骨骺分离 | 骨骺与干骺端移位 | 儿童长骨 |

## ❗ 高风险漏诊骨折提醒

1. **隐匿性骨折**：骨皮质断裂但无明显移位
   - 常见部位：耻骨支、坐骨支、小腕骨
   - 扫描策略：仔细检查骨皮质连续性

2. **应力性骨折**：早期无移位，仅有骨皮质不规则
   - 常见部位：第二、三跖骨、胫骨
   - 扫描策略：对比双侧对称骨骼

3. **疲劳性骨折**：骨膜反应、骨痂形成
   - 常见部位：腓骨下段、尺骨
   - 扫描策略：关注骨膜增厚区域

## 📝 报告输出格式

### 必须包含的字段：

1. **骨折位置**：具体骨骼名称 + 骨折部位（近端/中段/远端）
2. **骨折类型**：根据上述分类明确骨折类型
3. **骨折程度**：轻微/中度/重度（基于移位程度）
4. **是否移位**：是/否（如有移位，描述移位方向）
5. **并发症**：是否伴有关节脱位、软组织损伤

### 示例输出：
```
FINDINGS:

1. **右锁骨远端骨折**（粉碎性，伴移位）
   - 位置：右锁骨远端1/3
   - 类型：粉碎性骨折，3个骨碎片
   - 移位：骨折端分离约5mm，向上成角
   - 程度：中度
```

### 3.1.2 肺部病变专用Prompt（预计提升：10-15%识别率）

**核心思路**：
- 细化肺部病变分类
- 添加密度特征描述
- 添加位置定位指导

**优化Prompt示例**：
```markdown
# 肺部病变识别专用Prompt (CHEST X-RAY)

## 🔍 肺部病变识别步骤

### Step 1: 整体评估
1. 肺野是否清晰
2. 肺纹理是否增粗、紊乱
3. 肺门是否增大、模糊
4. 膈肌是否抬高、变平

### Step 2: 病变定位
1. 上叶、中叶、下叶
2. 内带、中带、外带
3. 前、中、后带

### Step 3: 病变性质判断
1. 密度：高密度、中等密度、低密度
2. 边缘：清晰、模糊、分叶、毛刺
3. 形状：圆形、类圆形、不规则形
4. 大小：测量直径或面积

## 📋 肺部病变识别清单

### ▶️ 肺实质病变
- [ ] 肺炎（大叶性、小叶性、间质性）
- [ ] 肺不张（_segments_、_segments_）
- [ ] 肺气肿（全小叶性、小叶中心性）
- [ ] 肺纤维化（蜂窝肺、牵拉性支扩）
- [ ] 肺实变（均匀/不均匀密度）

### ▶️ 肺间质病变
- [ ] 间质性肺炎
- [ ] 肺纤维化
- [ ] 肺水肿

### ▶️ 肺结节/肿块
- [ ] 单发结节（<3cm）
- [ ] 多发结节（>3个）
- [ ] 肿块（≥3cm）
- [ ] 空洞/空腔

### ▶️ 胸膜病变
- [ ] 胸腔积液（少量/中量/大量）
- [ ] 气胸
- [ ] 胸膜增厚

### ▶️ 纵隔病变
- [ ] 纵隔增宽
- [ ] 纵隔移位
- [ ] 纵隔肿块

## 🎯 肺部病变特征描述

### 肺炎特征
| 类型 | 密度 | 边缘 | 分布 | 特征 |
|-----|------|------|------|-----|
| 大叶性肺炎 | 均匀高密度 | 楔形边缘 | 整叶分布 | 支气管气象 |
| 小叶性肺炎 | 斑片状 | 模糊 | 两肺中下野 | 沿肺纹理分布 |
| 间质性肺炎 | 网状/粟粒状 | 清晰 | 两肺外带 | 肺纹理增粗 |

### 肺不张特征
| 类型 | 密度 | 边缘 | 体积 | 特征 |
|-----|------|------|------|-----|
| 完全性肺不张 | 均匀高密度 | 清晰 | 减小 | 纵隔移位 |
| 部分性肺不张 | 斑片状 | 模糊 | 局限 | 支气管征 |

### 肺气肿特征
| 类型 | 密度 | 边缘 | 分布 | 特征 |
|-----|------|------|------|-----|
| 全小叶性 | 低密度 | 模糊 | 散在分布 | 血管变细 |
| 小叶中心性 | 低密度 | 清晰 | 上叶分布 | 无血管征 |

## ❗ 高风险漏诊病变提醒

1. **隐匿性肺炎**：密度轻微增高，易被忽略
   - 扫描策略：对比双侧肺野密度

2. **局限性肺不张**：节段性不张，表现不典型
   - 扫描策略：关注肺纹理聚集区域

3. **小量胸腔积液**：积液量少，肋膈角变钝
   - 扫描策略：仔细观察肋膈角形态

4. **肺实变**：密度不均，易与肺炎混淆
   - 扫描策略：观察是否有支气管征、血管征

## 📝 报告输出格式

### 必须包含的字段：

1. **病变位置**：肺叶 + 肺段 + 内/中/外带
2. **病变性质**：根据上述分类明确病变类型
3. **病变大小**：测量或描述（小片状/大片状）
4. **密度特征**：均匀/不均匀、高/中/低密度
5. **边界特征**：清晰/模糊、分叶/毛刺
6. **伴随征象**：支气管气象、血管征、纵隔移位

### 示例输出：
```
FINDINGS:

1. **右肺下叶背段肺炎**（大叶性）
   - 位置：右肺下叶背段（后带）
   - 密度：均匀高密度
   - 边缘：楔形边缘
   - 范围：约5cm × 4cm
   - 特征：支气管气象阳性
```

### 3.2 方案二：后处理增强（推荐优先级：中）
